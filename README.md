# CustomJoinMessage

Этот плагин отправляет анимированное приветственное сообщение игроку при входе на сервер.

## Особенности

*   Анимированное сообщение (строки появляются поочередно).
*   Разные сообщения для новых и существующих игроков.
*   Поддержка кликабельных ссылок.
*   Интеграция с PlaceholderAPI и MiniPlaceholders (включая TAB).
*   Возможность отправки сообщения только игроку или всем на сервере.

## Установка

1.  Убедитесь, что на сервере установлены `PlaceholderAPI` и `MiniPlaceholders`.
2.  Скачайте `CustomJoinMessage-1.1.0.jar` из релизов.
3.  Поместите `.jar` файл в папку `plugins` вашего сервера Paper.
4.  Перезапустите сервер.
5.  Отредактируйте `config.yml` в папке `plugins/CustomJoinMessage/` под свои нужды.

## Конфигурация

См. `config.yml` после первого запуска плагина.
